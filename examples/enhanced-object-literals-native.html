Enhanced object literals (Native)

This page runs a series of a examples using enhanced object literals in ECMAScript 6 without any transpilation. This means your browser must support ES6 parameter handling syntax in order to successfully run the code. Open up your debugging console to view the output.

Browser support:

For more information on how enhanced object literals work and their level of support, check out the Enhanced object literals article that is part of the Learning ES6 series.
